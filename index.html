Cribl Search Pack for Zscaler: Your traffic went that way — now prove it | Cribl

Cribl released a Search Pack for Zscaler to simplify analysis of Zscaler security telemetry data.

  • The Search Pack provides pre-built dashboards tailored to different Zscaler log families (web, firewall, CASB, etc.).
  • Macros within the pack simplify search configuration by baking in dataset and sourcetype filters.
  • Datatypes are included to extract structured fields from Zscaler events, improving searchability.
  • The pack requires Zscaler logs to be ingested into a Cribl Search-compatible dataset (e.g., Cribl Lake, Lakehouse) and relies on either the Zscaler Stream Pack for Cribl Stream or pre-normalized logs for datatype application.

Cribl Search - Faster Log Investigations Without SIEM Costs or Complexity | Cribl

Cribl Search: AI-Powered Log Investigation

Cribl Search offers faster log investigations without the costs and complexity of a SIEM, leveraging AI to accelerate workflows and reduce manual toil.

  • Unified Platform: Combines data collection, normalization, storage, search, dashboards, and alerting into a single product.
  • AI-Assisted Investigation: Uses agentic AI and natural language queries to guide investigations, reducing the need for complex query building and accelerating time-to-resolution.
  • Search-in-Place Architecture: Eliminates data rehydration by searching data in its existing location (data lakes, object stores, etc.), reducing costs and delays.
  • Search Packs: Pre-built dashboards, searches, and configurations for common use cases (e.g., AWS WAF, Kubernetes) are available to accelerate onboarding and provide instant visibility.

Ocient Now Integrates with Datadog

Ocient now integrates directly with Datadog for simplified monitoring of large-scale data workloads.

  • The integration sends real-time metrics from Ocient Hyperscale Data Warehouse directly to Datadog.
  • Key metrics available include query performance, disk usage, table metrics (size, activity), and system health (CPU/memory).
  • The integration utilizes standard interfaces like SQL and JDBC for seamless communication.
  • Configuration is designed to be effortless with a guided setup process and customizable Datadog dashboards.

The Time is Now: Why Ocient is Your Netezza Replacement

Ocient positions itself as a direct replacement for IBM Netezza, particularly given IBM’s shift to a cloud-based Netezza offering and end-of-support for existing appliances in 2023.

  • The Ocient Hyperscale Data Warehouse (OHDW) is presented as a high-performance alternative built for modern enterprises, offering better storage efficiency and rapid analysis.
  • Ocient supports direct loading of various data formats – Avro, Parquet, JSON, and mainframe data – eliminating the need for separate ETL tooling.
  • The company offers a customer engagement model including no-cost pre-pilots and low-risk re-platforming pilots aiming for production deployment within months.
  • OHDW provides features not found in Netezza, including fully-managed service, multiple cloud deployment options, and dedicated migration support, with claims of up to 80% cost reduction.
